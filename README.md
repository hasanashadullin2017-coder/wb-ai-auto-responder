# AI-агент для Wildberries

Система автоматических ответов на отзывы и вопросы покупателей на Wildberries.

## Что делает система

AI-агент автоматически:

- получает новые отзывы через API Wildberries
- анализирует текст отзыва
- генерирует ответ с помощью OpenAI
- публикует ответ автоматически
- отвечает на вопросы покупателей
- сохраняет вопросы и ответы в базу знаний

## Возможности

- автоответы на отзывы
- ответы на вопросы покупателей
- подбор автоламп по автомобилю
- база знаний (RAG)
- автоматические допродажи
- интеграция с Wildberries API

## Технологии

- n8n
- OpenAI
- Perplexity
- Wildberries API
- Google Sheets
- Telegram API

## Архитектура

Wildberries API → n8n → AI обработка → публикация ответа → база знаний
